# Hello Universe - Python App

A simple Python application that displays "Hello Universe" on the screen.

## Installation

Make sure you have Python installed on your system.

Install dependencies:
```bash
pip install -r requirements.txt
```

## Usage

Run the application using one of the following commands:

**Basic version:**
```bash
python main.py
```

**Colorful version:**
```bash
python hello_universe.py
```

## What it does

This application prints "Hello Universe!" to the console with decorative formatting. The colorful version uses the `colorama` library to add colors to the output.

## Files

- `main.py` - Simple main application file
- `hello_universe.py` - Colorful version with colored output
- `requirements.txt` - Python dependencies
- `README.md` - This file
